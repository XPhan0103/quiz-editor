# Quiz Editor

A clean and minimal web application for creating, managing, and playing quizzes, built with React + Vite. This project allows you to easily build custom quizzes, add multiple-choice questions, take the tests yourself, and export them for use.

## Features

- **Create and update quizzes**: Easily build your own custom quizzes.
- **Manage questions**: Add, edit, remove, and reorder multiple-choice questions and their options.
- **Answer key**: Mark the correct answers for each question.
- **Preview & Play Mode**: Take the quiz in a beautifully designed play mode with instant, visual scoring.
- **Auto-save**: Automatically saves your work in progress to the browser's Local Storage.
- **Import / Export**: Save your quiz as a JSON file, or import an existing JSON quiz to continue editing.
- **Inline Validation**: Visually ensures questions have options and correct answers selected before saving or playing.

## Run locally

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```
