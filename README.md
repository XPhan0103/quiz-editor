# Quiz Editor

A clean and minimal web application for creating and managing quizzes, built with React + Vite. This project allows you to easily build custom quizzes, add multiple-choice questions, and export them for use.

## Features

- **Create and update quizzes**: Easily build your own custom quizzes.
- **Manage questions**: Add, edit, or remove multiple-choice questions and their options.
- **Answer key**: Mark the correct answers for each question.
- **Import / Export**: Save your quiz as a JSON file, or import an existing JSON quiz to continue editing.
- **Basic validation**: Ensures questions have options and correct answers selected.

## Run locally

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```
